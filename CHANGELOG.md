# Changelog

All notable changes to this project are documented in this file.

## [2026-04-29]

### Added
- Initial public version of **Rodičovský Kompas** orientational child-support calculator.
- Single-page flow with modal wizard for detailed care setup.
- Onboarding step with role selection (`maminka` / `tatínek`).
- Child entry with Czech name autocomplete and tag-style adding (max 3 children).
- Care split support across:
  - regular school-year routine,
  - summer holidays,
  - Christmas holidays,
  - spring holidays,
  - autumn holidays.
- Progress and completion feedback components.
- Project specification document (`SPEC.md`).

### Changed
- Rebranded app title and visual identity to **Rodičovský Kompas**.
- Reworked copy to calmer, parent-friendly language.
- Improved field ergonomics (input lengths, helper text, monthly CZK units).
- Result framing prioritizes parent perspective and clear payment direction.

### Notes
- Output is orientational and does not replace legal advice or court decision.
